bootstrap

主要就是用bootstrap里本来就有的内容

用git: npm install bootstrap (cnpm比较快)

表格：

<table>
    <thead>
        <th>表头的单元格</th>
    </thead>
    <tbody>
        <tr>  //表行
            <td>内容的单元格</td>
        </tr>
    </tbody>
</table>

图标的大小是font-size控制的，其他都一样

用before和after作左右边的线

h1:before{
    content:'';
    display:block;
    position:solid;
}
