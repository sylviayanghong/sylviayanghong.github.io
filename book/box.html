盒子模型（重要）

        *{
            margin: 0px;
            padding: 0px;
            box-sizing: border-box;

        }
        .box1{
            border: 10px solid #000;
            width: 200px;
            height: 200px;
            float: left;
            margin-right: 100px;
        }
        .box2{
            border: 10px solid #000;
            width: 200px;
            height: 200px;
            float: left;
        }
        .box1 h3{
            margin-left: 20px;
            margin-top: 50px;
        }
    </style>
</head>
<body>
    <div class="box1">
        <h3>盒子1</h3>
    </div>
    <div class="box2">
        <h3>盒子2</h3>
    </div>
</body>

audio---音乐标签

<audio src="" controls autoplay=""></audio>

<!-- controls是控制器面板属性，autoplay是自动播放属性，loop循环播放 -->

  • 块元素有文字的标签是让文字居中用text-algin：center；
  • 行内元素，行内块元素让父级text-algin:center;
