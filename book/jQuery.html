改变样式

注：js里''和""一样，一般代码里用'',组合用'你好"某某"'

$('选择器').css(xxx)
$('.box').css('bgc','blue')

//点击事件

$('.box').click(function(){
    $('.box').css('bgc','blue')
    })

改变属性

用.attr方法

<body>
    <div class="box">
        <img src="1.jpg" alt="">
    </div>

    <script src="jquery.js"></script>
    <script>

        // $('img').click(function(){
        //  $('img').attr('src','2.jpg')
        // })

        // 可以是父级也可以是自己

        $('.box').click(function(){
            $('img').attr('src','2.jpg')
        })
        $('img').click(function(){
            $('img').attr('src','3.jpg')
        })
        // 同时用时，box 变，img不变

    </script>
</body>
addClass 增加选择器操作样式比.css直接操作好
removeClass删除class
toggleClass切换class
    <style>
        .box{
            width: 100px;
            height: 100px;
            background-color: #0ff;
        }
        .active{
            width: 100px;
            height: 100px;
            background-color: #ff0;
        }
    </style>
</head>
<body>
    <div class="box active"></div>

    <script src='jquery.js'></script>

    <script>

        // $('.box').click(function(){
        //  $('.box').addClass('active')
        // })
        $('.box').click(function(){
            $('.box').removeClass('active')
        })
        // $('.box').click(function(){
        //  $('.box').toggleClass('active')
        // })

    </script>
siblings能找到同级的其他元素
    <style>
        .box{
            width: 100px;
            height: 100px;
            background-color: #0ff;
            transition: transform 3s;
        }
        .active{
            transform: translateX(500px);
        }
    </style>
</head>
<body>
    <div class="box"></div>
    <div class="box"></div>
    <div class="box"></div>
    <div class="box"></div>
    <div class="box"></div>
    <script src="jquery.js"></script>
    <script>

        $('.box').click(function(){
            $(this).toggleClass('active').siblings().removeClass('active');
        });

    </script>

里添加的得在div的class之后，不然div的class会覆盖增加的class

索引图

    <style>
        .btn li{
            width: 20px;
            height: 20px;
            border: 1px solid #000;
            list-style: none;
            float: left;
            text-align: center;
            line-height: 20px;
            cursor: pointer;
        }
    </style>
</head>
<body>
    <img class="pic" src="0.jpg" alt="">
    <ul class="btn">
        <li>1</li>
        <li>2</li>
        <li>3</li>
        <li>4</li>
        <li>5</li>
    </ul>

    <script src='jquery.js'></script>
    <script>

        $('.btn li').click(function(){
            var num = $(this).index();
            var picSrc = [
                '0.jpg',
                '1.jpg',
                '2.jpg',
                '3.jpg',
                '4.jpg'
                ];
            var mysrc = picSrc[num];
            $('.pic').attr('src',mysrc);
        });

    </script>
