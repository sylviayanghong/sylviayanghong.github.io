PS

  • ctrl+'+'/alt+滚轮:缩放
  • F8：信息
  • ctrl+r:标尺
  • 切片删除用delete
  • ctrl+0：显示整个页面/ctrl+1：1：1显示
  • 按住空格键拖动图片
图片存储
  • gif：动图，颜色单一（小）
  • jpg：颜色丰富（中）

  • png：透明度（大）

  • 要一行文字里前后颜色不一样，可加span

    <style>
  .h3{
      color: red;
  }
  .span{
      color: #000;
  }
</style>
<body>
  <h3><span>颜色</span>改变</h3>
</body>
