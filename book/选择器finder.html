选择器

选择器

1.元素选择器

<body>
    <h1>元素</h1>
</body>>
<style>
    h1{
    /*元素属性*/
}
</style>

2.id选择器

<body>
    <h3 id="hello">id</h3>
</body>
<style>
   #hello{
    /*元素属性*/
} 
</style>

3.类别（class）选择器

<body>
    <h3 class="world">id</h3>
</body>
<style>
   .world{
    /*元素属性*/
} 
</style>

4.层级选择器

<body>
    <ul class="fruit">
        <li></li>
    </ul>
</body>
<style>
   .fruit li{
    /*元素属性*/
} 
</style>

5.伪类选择器（用来指定一些动作）

<body>
    <ul class="fruit">
        <li></li>
    </ul>
</body>
<style>
   .fruit li{
        background-color: #fff;
} 
   .fruit li:hover{
     background-color: #f00;
   } 
</style>

6.*选择器可以选到所有

<style>
    *{
        margin: 0px;
        padding: 0px;
    } 
</style>
