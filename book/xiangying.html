HTML5

语义化的块元素
  1. section 局部区域 主要适用于移动端 比div大 一般包含div
  2. nav 导航栏 也是里边 ul>li
  3. hearder 头部
  4. footer 脚部
  5. aside 侧边栏
  6. article 文章
  7. title 标题
  8. main 主要区域 有且仅有一个 和h1标签一样 不能做为以上几种的子级 可以做父级
  9. 没有语义化的区域就可以用div
  10. lorem 添加文字快捷键

打断点

    <style>
        *{
            margin: 0px;
            padding: 0px;
        }
        .red{
            width: 200px;
            height: 400px;
            background-color: #f00;
            display: inline-block;
            /*少用浮动，用转为行内块元素的方式比较好，自己也要比较什么地方用什么较好*/
            /*去掉转块后中间的缝，把标签的回车换行去掉连接到一起就可以*/
        }
        .green{
            width: 200px;
            height: 200px;
            background-color: #0f0;
            display: inline-block;

        }
        /*断点处*/
        /*  screen桌面设备
            all所有
            print打印设备*/
        @media screen and (max-width: 768px){
            .green{
                background-color: #00f;
            }
        }
    </style>
</head>

布局断点

<style>
        *{
            margin: 0px;
            padding: 0px;
        }
        .header{
            width: 100%;
            height: 100px;
            background-color: #f00;
            text-align: center;
        }
        .nav{
            width: 20%;
            height: 400px;
            background-color: #0f0;
            text-align: center;
            /*用浮动比较好*/
            float: left;

        }
        .section{
            width: 60%;
            height: 400px;
            background-color: #00f;
            text-align: center;
            float: left;

        }

        .aside{
            width: 20%;
            height: 400px;
            background-color: #0ff;
            text-align: center;
            float: left;

        }
        .clear{
            clear: both;
        }
        .footer{
            width: 100%;
            height: 100px;
            background-color: #ff0;
            text-align: center;

        }
        /*断点处*/
        @media screen and (max-width: 640px){
            .nav{
                width: 100%;
                height: 100px;
                float: none;
            }
            .section{
                width: 100%;
                float: none;
            }
            .aside{
                width: 100%;
                height: 100px;
                float: none;
            }
        }
    </style>
</head>
<body>
    <header class="header">header</header>
    <main>
        <nav class="nav">nav</nav>
        <section class="section">section</section>
        <aside class="aside">aside</aside>
    </main>
    <div class="clear"></div>
    <footer class="footer">footer</footer>

</body>

        }
