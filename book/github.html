分支

当主分支上的内容需要修改但不能直接修改改变上线作品，可以建一个分支，在分支上修改然后合并两个分支内容，就不会影响页面内容

master是一个分支

git branch+分支名 建一个分支

git branch 查看分支

git checkout+分支名 切换分支

合并分支需要先切换到主分支然后用： git merge 分支名

主分支和次分支同时添加内容合并就会产生冲突，需要先切换到主分支，分支里的内容需要先提交然后合并

git branch -d 分支名 删除分支————只是删除本地分支

git push origin ：分支名 删除远程分支

github里也可操作，但涉及版本问题需先pull，建议都本地操作

  • mkdir+文件名 新建一个文件夹（目录）

  • 粘贴：shift+insert
  • cat+文件名：查看文件内容
  • 用户名+github+io：建自己的库发布到网络上
  • 在md里生成链接：链接名
